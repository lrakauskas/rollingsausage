This Privacy Policy applies to the Cartographers Map Generator mobile application published by Rolling Sausage.

Data Collection

Cartographers Map Generator does not collect, store, transmit, or share any personal information or user data.

The app does not collect information such as:

  • Personal details (name, email address, etc.)
  • Device identifiers
  • Location data
  • Usage analytics
  • Advertising identifiers

Local Data

Any data created while using the app (such as generated maps, scores, or settings) is stored only on your device and is never transmitted to the developer or any third party.

Third-Party Services

This app does not use:

  • Analytics services
  • Advertising networks
  • Cloud storage
  • External tracking tools

Data Sharing

Because no data is collected, no user data is shared with any third parties.

Children's Privacy

The application does not collect personal information from anyone, including children under 13.

Security

Since the app does not collect or transmit data, no personal information is stored on external servers.

Changes to This Privacy Policy

If this policy changes in the future, updates will be posted on this page with a revised date.
